# Interactive Analysis of Unemployment Causes in Bangladesh

## Description
This is a C++ console-based project that analyzes the major causes
of unemployment in Bangladesh using an interactive menu-driven system.

## How to Run the Project
1. Install a C++ compiler (GCC)
2. Open terminal or command prompt
3. Compile the code:
   g++ main.cpp -o unemployment
4. Run the program:
   ./unemployment
